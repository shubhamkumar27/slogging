# slogging

Sandbox of small projects. Hosted projects are served via GitHub Pages at
https://shubhamkumar27.github.io/slogging/.

## Layout

Each subdirectory is its own project. Some are hosted (static sites under
GitHub Pages), some are not (scripts, experiments, CLI tools).

## Adding a project

1. Create a subdirectory.
2. If it should be hosted, add an `index.html` and link it from the root
   `index.html`.
